# C++ Data Structures & Algorithms

A C++ implementation of fundamental data structures and algorithms, 
developed under Ubuntu as part of a programming skills validation project.

## Implemented Structures & Algorithms
- **Stack** – MakeEmpty, IsEmpty, IsFull, Push, Pop, DisplayStack
- **Queue** – MakeEmpty, IsEmpty, IsFull, Enqueue, Dequeue, DisplayQueue
- **Merge Sort**
- **Binary Search**
- **Binary Tree Traversals** – Preorder, Inorder, Postorder
- **Graph Traversals** – Depth-First (DFS) and Breadth-First (BFS)

## How to Compile & Run
```bash
g++ main.cpp -o program
./program
```

## Environment
- Language: C++
- OS: Ubuntu (Linux)
